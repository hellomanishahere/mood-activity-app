# 🧠 Mental Health Activity Recommender

## 📌 Project Description

🚀 A web app that suggests activities based on user mood to improve mental well-being.

This web application helps users discover activities based on their current mood. It suggests simple and engaging activities that can improve mental well-being using a public API.

---

## 🎯 Purpose

This project is built to demonstrate:

* JavaScript fundamentals
* API integration using fetch
* Use of array higher-order functions (map, filter, sort)
* Responsive UI design

---

## 🌐 API Used
- GitHub REST API  
  https://api.github.com/search/users?q=example

---

## ✨ Planned Features

* 🎭 Mood selection (happy, sad, stressed, bored)
* 🎯 Activity suggestions based on mood
* 🔍 Search activities by type
* 🎛️ Filter activities (participants, category)
* 🔃 Sort activities
* ❤️ Add to favorites
* 🌙 Dark mode toggle

---

## 🛠️ Technologies Used

* HTML
* CSS (or Tailwind/Bootstrap)
* JavaScript (ES6+)
* Fetch API

---

## 🚀 How to Run the Project
